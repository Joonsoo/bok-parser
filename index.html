Moon Parser

Download .zip Download .tar.gz View on GitHub

Moon Parser is a parser that supports languages with more flexible structure and richer idioms, along with laugnage analysis tools.

Moon Parser is featuring dynamic parsing, which proceeds parsing and language analysis simultaneously. Moon Parser also includes visualization tools to show the given languages, parsed results, and parsing procedures. And I am planning to implement static language analyzer and parser generator.
